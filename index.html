Module 02 — pourquoi un dictionnaire, et pas une liste, pour représenter des produits.

boulangerie_lea.py
def calculer_commande(nb_croissants, nb_pains):
    prix = nb_croissants * 1.20 + nb_pains * 0.80
    nb_total = nb_croissants + nb_pains

    if nb_total > 3:      # règle de promo
        prix = prix * 0.90
    return prix

print(calculer_commande(3, 2))
print(calculer_commande(5, 1))

Module 05 — une seule fonction pour cent commandes, une seule ligne à changer si la promo évolue.

calcul_age_protege.py
try:
    annee = int(input("Année de naissance ? "))
    age = 2026 - annee
    print(f"Tu as {age} ans.")
except ValueError:
    print("Oups ! Entre une année en chiffres, pas des lettres.")
